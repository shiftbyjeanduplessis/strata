# STRATA v1.5.11 — Separate Mobile/Desktop Pages

Frozen scope remains unchanged. This pass separates the mobile execution page and the desktop planning/builder page.

## Pages

- `index.html` — mobile app: Today, Training, Body, Cardio logging, Account.
- `desktop.html` — desktop builder/planner: lifting builder and cardio prescription planner.

## Rule

Mobile is for execution. Desktop is for planning. The builder is no longer injected into the mobile Today page.

## Still included

- Active workout protection
- Builder publish validation
- Honest progression rules
- Setup/cue visibility in mobile exercise focus
- Cardio kept outside lifting
- Supabase sync/account backup

No new product features were added.
